# hShop DS

Nintendo DS/GBA game downloader over Wi-Fi. No PC needed.

The hShop NDS was originally created for Twilight Menu but works on Luma using "Nds Forwarder Generator" 
(download games to your SD card, then install them to the 3DS menu with the generator).

## Features

- Browse 78+ NDS/GBA titles
- Direct download to SD card
- Resume support for large files
- Dual-screen interface (game list + download progress)

## Requirements

- DS/DSi/3DS with custom firmware or flashcard
- Wi-Fi connection (WEP or open network)(WPA2 only for Luma)
- SD card with enough space

## Installation

1. Download the .nds needed
2. Copy to SD card root
3. Launch via TWiLight Menu++ or NDS Forwarder

**3DS users:** Configure DS Wi-Fi settings separately if you're using a R4 (not the 3DS Wi-Fi settings)

## Usage

- **D-Pad**: Navigate game list
- **A**: Download selected game
- **B**: Cancel/Exit

## Known Issues

- Large files (500MB+) take 1 hour or more on DS Phat/Lite
- Connection drops may require restart
- WPA2 networks not supported (hardware limitation on DS)
